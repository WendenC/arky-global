# Assets - Imagens e SVGs

Esta pasta contém todas as imagens e SVGs usados na landing page.

## Como obter os assets do Figma

1. **Acesse o Figma Desktop** e abra o design: `LP---ArkyGlobal`

2. **Para baixar imagens PNG:**
   - Selecione o elemento no Figma
   - Clique com botão direito → "Export" ou use Ctrl+Shift+E
   - Escolha formato PNG
   - Salve na pasta `public/assets/` com o nome correto

3. **Para baixar SVGs:**
   - Selecione o elemento no Figma
   - Clique com botão direito → "Copy as SVG" ou "Export" como SVG
   - Salve na pasta `public/assets/` com o nome correto
